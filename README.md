# 翠巒祭会計システム

## 概要
高崎高校主催の学園祭「翠巒祭」の販売関係で使用する会計システム

## 技術
### 店舗向けアプリケーション
```
デザイン/プログラム: Dart?
フレームワーク: Flutter?
```

### お客様向けアプリケーション
```
デザイン: HTML/CSS
プログラム: C#(.NET8)
フレームワーク: Blazor
```

**動作方法**
```shell
$ cd suiran
# dotnetコマンドをインストール
$ dotnet watch run --pathbase=/suiran
# http://localhost:5291/suiran にブラウザでアクセス
```
